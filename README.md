# Mobile Sales Dashboard

## Project Overview
This project is an interactive Power BI dashboard developed to analyze mobile sales performance. It provides business insights through interactive reports, KPIs, and visualizations.

## Tools Used
- Power BI
- Power Query
- DAX
- Microsoft Excel

## Report Pages

### 1. Dashboard
- Total Sales
- Total Quantity Sold
- Total Transactions
- Average Price
- Sales by City
- Monthly Sales Trend
- Sales by Brand
- Sales by Mobile Model
- Payment Method Analysis
- Customer Rating Analysis

### 2. MTD (Month-to-Date) Report
- Current month sales performance
- MTD revenue analysis
- MTD transaction analysis
- MTD quantity sold
- Performance tracking with interactive filters

### 3. Same Period Last Year (SPLY)
- Comparison of current performance with the same period last year
- Year-over-Year (YoY) sales comparison
- Growth and decline analysis
- Performance trend analysis
